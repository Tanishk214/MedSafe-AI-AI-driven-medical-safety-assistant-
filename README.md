# MedSafe-AI


## Description

A smart healthcare project that helps users detect medical issues and provides AI-based suggestions for safer decisions.

## Problem Statement
Many people lack quick access to reliable medical guidance. This project aims to provide basic assistance using AI.


## Features
- User-friendly interface
- AI-based prediction
- Fast results
- Secure data handling

## Tech Stack
- Python
- Streamlit
- Pandas / NumPy 

## Installation and How to run

1. Clone the repository
   git clone <repo-link>

2. Go to project folder
   cd project-folder

3. Install dependencies
   pip install -r requirements.txt

4. Run the app
   streamlit run app.py
